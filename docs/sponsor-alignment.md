# Sponsor Alignment

## Primary Prize Targets

### 0G

Use 0G for decentralized storage of execution receipts, task artifacts, and agent memory references. The MVP should make 0G visible in the core flow, not only as a hidden upload step.

### Gensyn AXL

Use AXL for agent-to-agent task dispatch and response. The demo should show at least two independent agent participants exchanging task and result messages.

### KeeperHub

Use KeeperHub for reliable execution, onchain follow-through, recurring verification, or receipt anchoring. The integration should be part of the final task lifecycle.
